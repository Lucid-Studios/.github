Privacy Policy for Lucid Action Hub

Effective date: August 16, 2025

Lucid Technologies (“we”, “our”, “us”) operates the Lucid Action Hub, which provides AI Actions to our teams.

1. Data Collection

  • Text inputs sent to Actions

2. Data Use

  • To fulfill requested Actions

3. Data Retention

Logs retained up to 30 days; aggregated anonymized metrics may be stored longer.

4. Security

HTTPS in transit, restricted access to logs, secrets not stored in plain text.

5. User Rights

Email contact@lucidTechnologies.tech to request deletion of logs within the retention window or ask questions.

We may update this policy; the latest version will always appear on this page.
