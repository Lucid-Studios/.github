# Code of Conduct

Lucid Studios expects respectful, evidence-based collaboration across its repositories.

## Expected Behavior

- communicate clearly and professionally
- critique ideas and implementations rather than people
- document assumptions, risks, and uncertainties honestly
- preserve scientific, technical, and provenance integrity
- respect security, privacy, and data-handling boundaries

## Unacceptable Behavior

- harassment, intimidation, discrimination, or personal attacks
- intentional falsification of code, tests, provenance, results, or documentation
- publication of secrets, private keys, restricted data, or sensitive vulnerability details
- malicious, deceptive, or counterfeit contributions

## Scope

This code of conduct applies to Lucid Studios GitHub collaboration spaces, including issues, pull requests, code review, and governance discussions.

## Reporting

If public reporting would be inappropriate, use the private security or maintainer contact path described in [`SECURITY.md`](SECURITY.md).
