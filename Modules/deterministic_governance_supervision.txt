\begin{abstract}
This document outlines a provisional framework for constructing governance-aware
training datasets derived from deterministic system traces rather than raw conversational corpora.
The approach separates probabilistic proposal generation from structural admissibility,
privilege governance, and identity continuity. The objective is not to replace
machine learning, but to strengthen it through typed, witnessed, and auditable
supervision surfaces.
\end{abstract}

\section{Motivation}

Traditional model training relies on large, blended corpora that implicitly encode
behavioral norms, domain knowledge, and policy constraints within model weights.
This approach has several limitations:

\begin{itemize}
    \item Lack of modal isolation between domains.
    \item Blended policy learning without explicit traceability.
    \item Limited auditability of decision provenance.
    \item Difficulty separating proposal competence from governance competence.
\end{itemize}

We propose an alternative supervisory surface derived from deterministic governance traces.
Rather than training on raw conversation logs, we train on structured,
canonicalized governance events.

\section{Architectural Separation}

We distinguish three layers:

\begin{enumerate}
    \item \textbf{Proposer Layer (LLM)} -- Generates candidate structured proposals.
    \item \textbf{Governance Layer (SLI + SAT + Mount Discipline)} -- Determines admissibility and privilege.
    \item \textbf{Continuity Layer (Engrammitization)} -- Advances identity via witnessed commitments.
\end{enumerate}

Training occurs only at the proposer layer.
Governance and continuity remain deterministic.

\section{Witnessed-Morphism Dataset Concept}

Instead of training on blended natural language corpora,
we construct datasets from typed governance events:

\subsection{Ingestion Supervision}

Each record contains:

\begin{itemize}
    \item Input: RawDescriptor
    \item Output: IngestionOutcome $\in \{OK, NEEDS\_SPEC, REJECT\}$
    \item MissingFields (ordered)
    \item ReasonCode (stable constant)
\end{itemize}

This trains structural admissibility repair rather than domain facts.

\subsection{SLI Governance Supervision}

Each record contains:

\begin{itemize}
    \item Input: (Intent, SessionStateSummary)
    \item Output: SliDecision (ALLOW/DENY)
    \item ReasonCode
    \item MountPresent flag
    \item SAT mode requirement
\end{itemize}

This trains correct handle selection and privilege awareness.

\subsection{Elevation Negotiation Supervision}

Each record contains:

\begin{itemize}
    \item Input: Denial reason (e.g., SAT\_INSUFFICIENT)
    \item Output: ElevationRequestForm or FallbackStrategy
\end{itemize}

This teaches explicit privilege mobility patterns.

\section{Continuity and Compression}

Committed events generate:

\begin{itemize}
    \item Theater-local identity references
    \item Witness event sets
    \item NormalFormKey (deterministic equivalence key)
\end{itemize}

NormalFormKey functions as a structural compression handle,
allowing equivalence classes of morphisms to collapse without
storing all derived surfaces.

\section{Key Distinction}

This approach does not eliminate machine learning.
Instead, it relocates specialization pressure:

\begin{itemize}
    \item ML handles proposal distribution.
    \item Governance enforces admissibility.
    \item Continuity preserves witnessed commitments.
\end{itemize}

The dataset trains models to operate competently within deterministic governance surfaces,
rather than embedding domain behavior implicitly in weights.

\section{Research Questions}

\begin{enumerate}
    \item How much domain specialization can be shifted from parameter space
          into governance constraint space?
    \item Can admissibility training reduce retry loops without domain fine-tuning?
    \item Does modal isolation reduce cross-domain behavioral bleed?
    \item What compression ratios emerge under NormalFormKey equivalence?
\end{enumerate}

\section{Status}

This document defines a future research direction.
Implementation depends on completion of deterministic
engrammitization and theater-local identity formation.
