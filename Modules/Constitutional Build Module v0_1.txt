Alright — here’s a tight **one-page spec** for the Constitutional Build Module (CBM) as we’ve framed it.

# Constitutional Build Module v0.1 — One-Page Spec

## Purpose

Make **Frame the invariant** and treat generation as **predicate outflow** by enforcing a compiled constitution (contracts + obligations + validators) over any model output, with deterministic telemetry for research.

CBM is **not** an LLM. It is a **governance + audit + rewrite harness** that can wrap any generator.

---

## Core Concepts

### Modes

* **AUDIT**: no silent repair; must prove/disprove/inconsistent; no hooks/questions.
* **SPEC**: definitions → lemmas → proof sketch → diagnostics; no metaphors/hooks.
* **IDEATE**: exploration allowed; hooks optional; must label speculation vs claim.

### Constitution

A **FrameContract** compiled from intent + mode + assumptions that defines:

* what is allowed
* what must be produced
* what is forbidden
* how violations are detected
* how rewrites are directed

---

## Module Boundary Diagram

**Inputs** → **CBM** → **Outputs**

* Intent / Assumptions / Context → **ContractCompiler**
* Candidate text / structured output → **Validators**
* Violations → **RewriteDirector**
* Accepted artifact → **Emitter + TelemetrySink**

CBM wraps a **Generator** (external) but never trusts it.

---

## v0.1 Components

### 1) ContractCompiler

**Input:** `IntentForm`, `FrameMode`, `AssumptionSet`, `PolicyProfile`
**Output:** `FrameContract`

Responsibilities:

* Normalize assumptions (canonical ordering, stable IDs)
* Compile obligations (must end PROVED/DISPROVED/INCONSISTENT, etc.)
* Compile forbidden lexicon + structural constraints per mode
* Produce deterministic `contractId = SHA256(canonical_contract_bytes)`

### 2) Validators

**Input:** `FrameContract`, `CandidateOutput`
**Output:** `ViolationReport` (zero or more violations)

Validator families (v0.1 = rule-based + deterministic):

* **Hook/Voice Gate** (AUDIT/SPEC):

  * no trailing question marks
  * no invitation phrases (“would you like”, “shall we”, etc.)
  * optional: no 1st/2nd person pronouns
* **Metaphor Gate** (AUDIT/SPEC):

  * lexicon-based (configurable)
* **Proof Obligation Gate**:

  * must conclude with exactly one: `PROVED | DISPROVED | INCONSISTENT | UNDEFINED`
  * if `INCONSISTENT`, must include `MinimalConflictingSubset`
* **Assumption Integrity Gate** (“no silent repair”):

  * detect weakening/rewrites like:

    * “for all” → “in expectation”
    * “full-rank” → “degenerate”
    * “must” → “can”
  * must raise `ASSUMPTION_DRIFT` if altered without explicit disclosure marker

### 3) RewriteDirector

**Input:** `FrameContract`, `ViolationReport`
**Output:** `RewriteDirective`

Responsibilities:

* Emit *minimal* rewrite instruction (no new solution content)
* Enforce a rewrite protocol per mode, e.g.:

  * “Remove all questions, remove invitations, preserve equations, end DISPROVED + minimal conflicting subset.”
* Deterministic `directiveId` based on contract + violations

### 4) Run Orchestrator

**Input:** generator callback + CBM components
**Output:** accepted output or bounded failure

Workflow:

1. Compile contract
2. Generate candidate
3. Validate
4. If violations:

   * issue rewrite directive
   * retry up to `maxRewritePasses` (config)
5. Accept or fail with structured reason

### 5) TelemetrySink (append-only, deterministic)

NDJSON record per attempt + per accepted emission.

---

## Data Structures (Minimal)

### `FrameContract`

* `contractId`
* `mode` (AUDIT/SPEC/IDEATE)
* `assumptions[]` (canonical: id, text, quantifiers, tags)
* `obligations[]` (enum + params)
* `forbiddenLexicon[]`
* `requiredSections[]` (SPEC mode)
* `maxRewritePasses`
* `policyProfileId`

### `CandidateOutput`

* `rawText`
* `structured` (optional: sections, claims, proof blocks)
* `hash`

### `ViolationReport`

* `violations[]` each:

  * `code` (HOOK, METAPHOR, ASSUMPTION_DRIFT, MISSING_OBLIGATION, etc.)
  * `span` (start/end indices) if applicable
  * `details` (machine-readable)
* `severity` summary
* `suggestedActions[]` (optional)

### `RewriteDirective`

* `directiveId`
* `instructions` (minimal)
* `requiredConclusion` (optional)
* `forbiddenPatterns` (optional)

### `RunRecord` (Telemetry)

* `runId` (deterministic)
* `configId` (hash of knob settings)
* `contractId`
* `attemptIndex`
* `candidateHash`
* `violationsHash`
* `accepted` bool
* `rewritePassesUsed`
* `metrics` (below)

## Metrics (v0.1)

* `HookRate` = hooks / tokens (or per 1k chars)
* `MetaphorRate`
* `AssumptionDriftRate`
* `ObligationPassRate`
* `RewriteCycles`
* `AcceptanceRate`
* `FlowScore` (simple v0.1):

  * `FlowScore = AcceptanceRate - 0.1*RewriteCycles - 0.2*AssumptionDriftRate - 0.2*HookRate`

---

## Acceptance Criteria (v0.1)

CBM is “working” when:

1. In **AUDIT** mode, outputs:

* contain no hooks/invitations/questions (per validator)
* never silently repair assumptions
* always end with an allowed conclusion token

2. Under contradictory assumption sets, CBM can force:

* `INCONSISTENT` + minimal conflicting subset
* **without** generator inventing fixes

3. Telemetry is:

* deterministic
* append-only
* sufficient to reproduce config + violations + acceptance path

```csharp
using System;
using System.Collections.Generic;

namespace Oan.Core.Constitution
{
    public enum FrameMode { Audit, Spec, Ideate }

    public enum ConclusionKind { Proved, Disproved, Inconsistent, Undefined }

    public enum ViolationCode
    {
        Hook,
        Metaphor,
        MissingObligation,
        InvalidConclusion,
        AssumptionDrift,
        InconsistentMissingMinimalSubset,
        ForbiddenPattern
    }

    public sealed record Assumption(
        string id,
        string text,
        string? quantifier = null,   // e.g. "FOR_ALL", "EXISTS", "IN_EXPECTATION"
        string? tags = null          // optional, e.g. "noise", "invariant"
    );

    public sealed record Obligation(
        string code,                 // e.g. "MUST_HAVE_CONCLUSION"
        string? param = null         // e.g. "AUDIT"
    );

    public sealed record FrameContract(
        string contractId,
        FrameMode mode,
        IReadOnlyList<Assumption> assumptions,
        IReadOnlyList<Obligation> obligations,
        IReadOnlyList<string> forbiddenLexicon,
        IReadOnlyList<string> requiredSections,
        int maxRewritePasses,
        string policyProfileId
    );

    public sealed record CandidateOutput(
        string rawText,
        string hashHexLower
    );

    public sealed record Violation(
        ViolationCode code,
        int? spanStart = null,
        int? spanEnd = null,
        string? details = null
    );

    public sealed record ViolationReport(
        IReadOnlyList<Violation> violations,
        string violationsHashHexLower,
        int severity // simple aggregate
    )
    {
        public bool IsClean => violations.Count == 0;
    }

    public sealed record RewriteDirective(
        string directiveId,
        string instructions,
        ConclusionKind? requiredConclusion = null,
        IReadOnlyList<string>? forbiddenPatterns = null
    );

    public sealed record RunMetrics(
        double hookRate,
        double metaphorRate,
        double assumptionDriftRate,
        double obligationPassRate,
        int rewriteCycles,
        bool accepted,
        double flowScore
    );

    public sealed record RunRecord(
        string runId,
        string configId,
        string contractId,
        int attemptIndex,
        string candidateHash,
        string violationsHash,
        bool accepted,
        int rewritePassesUsed,
        RunMetrics metrics,
        long tickUtc // for logging only; do NOT include in ids/hashes
    );
}
```

## 2) Deterministic hashing + canonical serialization

```csharp
using System;
using System.Collections.Generic;
using System.Globalization;
using System.Security.Cryptography;
using System.Text;

namespace Oan.Core.Constitution
{
    public static class Determinism
    {
        public static string Sha256HexLower(ReadOnlySpan<byte> bytes)
        {
            using var sha = SHA256.Create();
            var hash = sha.ComputeHash(bytes.ToArray());
            return Convert.ToHexString(hash).ToLowerInvariant();
        }

        public static string Sha256HexLower(string s)
            => Sha256HexLower(Encoding.UTF8.GetBytes(s));

        // Canonical: explicit field ordering, \n line endings, ordinal sorting where needed.
        public static string CanonicalizeContract(FrameContract draft)
        {
            var sb = new StringBuilder(4096);

            sb.Append("mode=").Append(draft.mode.ToString()).Append('\n');
            sb.Append("policyProfileId=").Append(draft.policyProfileId).Append('\n');
            sb.Append("maxRewritePasses=").Append(draft.maxRewritePasses.ToString(CultureInfo.InvariantCulture)).Append('\n');

            // Assumptions: stable ordering by id (ordinal)
            var assumptions = new List<Assumption>(draft.assumptions);
            assumptions.Sort((a, b) => StringComparer.Ordinal.Compare(a.id, b.id));

            sb.Append("assumptions.count=").Append(assumptions.Count.ToString(CultureInfo.InvariantCulture)).Append('\n');
            foreach (var a in assumptions)
            {
                sb.Append("assumption.id=").Append(a.id).Append('\n');
                sb.Append("assumption.quantifier=").Append(a.quantifier ?? "null").Append('\n');
                sb.Append("assumption.tags=").Append(a.tags ?? "null").Append('\n');
                sb.Append("assumption.text=").Append(a.text).Append('\n');
                sb.Append("--\n");
            }

            // Obligations: stable ordering by code then param
            var obligations = new List<Obligation>(draft.obligations);
            obligations.Sort((x, y) =>
            {
                int c = StringComparer.Ordinal.Compare(x.code, y.code);
                if (c != 0) return c;
                return StringComparer.Ordinal.Compare(x.param ?? "", y.param ?? "");
            });

            sb.Append("obligations.count=").Append(obligations.Count.ToString(CultureInfo.InvariantCulture)).Append('\n');
            foreach (var o in obligations)
            {
                sb.Append("obligation.code=").Append(o.code).Append('\n');
                sb.Append("obligation.param=").Append(o.param ?? "null").Append('\n');
                sb.Append("--\n");
            }

            // Forbidden lexicon: stable ordering ordinal
            var forbidden = new List<string>(draft.forbiddenLexicon ?? Array.Empty<string>());
            forbidden.Sort(StringComparer.Ordinal);
            sb.Append("forbidden.count=").Append(forbidden.Count.ToString(CultureInfo.InvariantCulture)).Append('\n');
            foreach (var w in forbidden) sb.Append("forbidden=").Append(w).Append('\n');

            // Required sections: stable ordering ordinal (or keep as given—choose one; I’ll sort to be safe)
            var sections = new List<string>(draft.requiredSections ?? Array.Empty<string>());
            sections.Sort(StringComparer.Ordinal);
            sb.Append("sections.count=").Append(sections.Count.ToString(CultureInfo.InvariantCulture)).Append('\n');
            foreach (var s in sections) sb.Append("section=").Append(s).Append('\n');

            return sb.ToString();
        }

        public static string CanonicalizeViolations(IReadOnlyList<Violation> violations)
        {
            var list = new List<Violation>(violations);
            list.Sort((a, b) =>
            {
                int c = a.code.CompareTo(b.code);
                if (c != 0) return c;
                c = (a.spanStart ?? -1).CompareTo(b.spanStart ?? -1);
                if (c != 0) return c;
                c = (a.spanEnd ?? -1).CompareTo(b.spanEnd ?? -1);
                if (c != 0) return c;
                return StringComparer.Ordinal.Compare(a.details ?? "", b.details ?? "");
            });

            var sb = new StringBuilder(1024);
            sb.Append("violations.count=").Append(list.Count.ToString(CultureInfo.InvariantCulture)).Append('\n');
            foreach (var v in list)
            {
                sb.Append("code=").Append(v.code.ToString()).Append('\n');
                sb.Append("start=").Append((v.spanStart ?? -1).ToString(CultureInfo.InvariantCulture)).Append('\n');
                sb.Append("end=").Append((v.spanEnd ?? -1).ToString(CultureInfo.InvariantCulture)).Append('\n');
                sb.Append("details=").Append(v.details ?? "null").Append('\n');
                sb.Append("--\n");
            }
            return sb.ToString();
        }
    }
}
```

## 3) ContractCompiler

```csharp
using System;
using System.Collections.Generic;

namespace Oan.Core.Constitution
{
    public sealed class ContractCompiler
    {
        public FrameContract Compile(
            FrameMode mode,
            IReadOnlyList<Assumption> assumptions,
            string policyProfileId,
            int maxRewritePasses,
            IReadOnlyList<string>? forbiddenLexicon = null,
            IReadOnlyList<string>? requiredSections = null)
        {
            // Obligations by mode
            var obligations = new List<Obligation>
            {
                new Obligation("MUST_HAVE_CONCLUSION", mode.ToString().ToUpperInvariant())
            };

            if (mode == FrameMode.Audit || mode == FrameMode.Spec)
            {
                obligations.Add(new Obligation("NO_HOOKS"));
                obligations.Add(new Obligation("NO_SILENT_REPAIR"));
            }

            if (mode == FrameMode.Spec)
            {
                obligations.Add(new Obligation("REQUIRE_SECTIONS", "DEFINITIONS,LEMMAS,PROOF,DIAGNOSTICS"));
            }

            var draft = new FrameContract(
                contractId: "DRAFT",
                mode: mode,
                assumptions: assumptions ?? Array.Empty<Assumption>(),
                obligations: obligations,
                forbiddenLexicon: forbiddenLexicon ?? DefaultForbiddenLexicon(mode),
                requiredSections: requiredSections ?? DefaultRequiredSections(mode),
                maxRewritePasses: maxRewritePasses,
                policyProfileId: policyProfileId
            );

            string canonical = Determinism.CanonicalizeContract(draft);
            string id = Determinism.Sha256HexLower(canonical);

            return draft with { contractId = id };
        }

        private static IReadOnlyList<string> DefaultForbiddenLexicon(FrameMode mode)
        {
            if (mode == FrameMode.Ideate) return Array.Empty<string>();
            // Keep v0.1 tiny; expand later
            return new[]
            {
                "would you like",
                "shall we",
                "i'm curious",
                "i am curious",
                "sovereign",
                "feel",
                "crystal",
                "shatter",
                "melting"
            };
        }

        private static IReadOnlyList<string> DefaultRequiredSections(FrameMode mode)
        {
            if (mode != FrameMode.Spec) return Array.Empty<string>();
            return new[] { "Definitions", "Lemmas", "Proof", "Diagnostics" };
        }
    }
}
```

### 4a) Utility parsing helpers

```csharp
using System;
using System.Collections.Generic;
using System.Text.RegularExpressions;

namespace Oan.Core.Constitution
{
    internal static class TextScan
    {
        public static bool EndsWithQuestion(string text)
        {
            if (string.IsNullOrWhiteSpace(text)) return false;
            for (int i = text.Length - 1; i >= 0; i--)
            {
                char c = text[i];
                if (char.IsWhiteSpace(c)) continue;
                return c == '?';
            }
            return false;
        }

        public static IEnumerable<(int start, int end)> FindAllOrdinalIgnoreCase(string haystack, string needle)
        {
            if (string.IsNullOrEmpty(haystack) || string.IsNullOrEmpty(needle)) yield break;

            int idx = 0;
            while (true)
            {
                idx = haystack.IndexOf(needle, idx, StringComparison.OrdinalIgnoreCase);
                if (idx < 0) yield break;
                yield return (idx, idx + needle.Length);
                idx += needle.Length;
            }
        }

        public static bool ContainsOrdinalIgnoreCase(string haystack, string needle)
            => haystack?.IndexOf(needle, StringComparison.OrdinalIgnoreCase) >= 0;
    }
}
```

### 4b) IValidator interface + composite runner

```csharp
using System.Collections.Generic;

namespace Oan.Core.Constitution
{
    public interface IValidator
    {
        void Validate(FrameContract contract, string candidateText, List<Violation> violations);
    }

    public sealed class CompositeValidator
    {
        private readonly IReadOnlyList<IValidator> _validators;

        public CompositeValidator(IReadOnlyList<IValidator> validators)
        {
            _validators = validators;
        }

        public ViolationReport Validate(FrameContract contract, string candidateText)
        {
            var violations = new List<Violation>(16);

            foreach (var v in _validators)
                v.Validate(contract, candidateText, violations);

            string vCanon = Determinism.CanonicalizeViolations(violations);
            string vHash = Determinism.Sha256HexLower(vCanon);

            int severity = 0;
            foreach (var x in violations)
            {
                severity += x.code switch
                {
                    ViolationCode.AssumptionDrift => 3,
                    ViolationCode.InvalidConclusion => 3,
                    ViolationCode.MissingObligation => 2,
                    ViolationCode.InconsistentMissingMinimalSubset => 3,
                    ViolationCode.Hook => 1,
                    ViolationCode.Metaphor => 1,
                    _ => 1
                };
            }

            return new ViolationReport(violations, vHash, severity);
        }
    }
}
```

### 4c) Hook/Voice validator

```csharp
using System.Collections.Generic;

namespace Oan.Core.Constitution
{
    public sealed class HookValidator : IValidator
    {
        public void Validate(FrameContract contract, string candidateText, List<Violation> violations)
        {
            if (contract.mode == FrameMode.Ideate) return;

            if (TextScan.EndsWithQuestion(candidateText))
            {
                violations.Add(new Violation(ViolationCode.Hook, details: "Output ends with a question mark."));
            }

            // Invitation phrases (lexicon-based)
            foreach (var phrase in contract.forbiddenLexicon)
            {
                foreach (var (s, e) in TextScan.FindAllOrdinalIgnoreCase(candidateText, phrase))
                {
                    violations.Add(new Violation(ViolationCode.Hook, s, e, $"Forbidden phrase: '{phrase}'"));
                }
            }
        }
    }
}
```

### 4d) Obligation validator (conclusion + minimal conflicting subset)

This is lightweight; you can later enforce a full schema.

```csharp
using System;
using System.Collections.Generic;

namespace Oan.Core.Constitution
{
    public sealed class ObligationValidator : IValidator
    {
        public void Validate(FrameContract contract, string candidateText, List<Violation> violations)
        {
            // Must have explicit conclusion token anywhere (v0.1). Later enforce last line only.
            var kind = TryDetectConclusion(candidateText);

            if (kind == null)
            {
                violations.Add(new Violation(ViolationCode.MissingObligation, details: "Missing conclusion token: PROVED/DISPROVED/INCONSISTENT/UNDEFINED"));
                return;
            }

            if (kind == ConclusionKind.Inconsistent)
            {
                // Require phrase "Minimal Conflicting Subset" (v0.1 marker)
                if (!TextScan.ContainsOrdinalIgnoreCase(candidateText, "minimal conflicting subset"))
                {
                    violations.Add(new Violation(ViolationCode.InconsistentMissingMinimalSubset, details: "INCONSISTENT requires 'Minimal Conflicting Subset' section/marker."));
                }
            }
        }

        private static ConclusionKind? TryDetectConclusion(string text)
        {
            // v0.1: simple contains checks (case-insensitive)
            if (TextScan.ContainsOrdinalIgnoreCase(text, "PROVED")) return ConclusionKind.Proved;
            if (TextScan.ContainsOrdinalIgnoreCase(text, "DISPROVED")) return ConclusionKind.Disproved;
            if (TextScan.ContainsOrdinalIgnoreCase(text, "INCONSISTENT")) return ConclusionKind.Inconsistent;
            if (TextScan.ContainsOrdinalIgnoreCase(text, "UNDEFINED")) return ConclusionKind.Undefined;
            return null;
        }
    }
}
```

### 4e) Assumption drift validator (detect silent weakening)

v0.1: crude but effective (pattern-based). Later you can do AST-level parsing.

```csharp
using System.Collections.Generic;

namespace Oan.Core.Constitution
{
    public sealed class AssumptionDriftValidator : IValidator
    {
        // Common “repair” phrases that often indicate weakening
        private static readonly string[] DriftMarkers =
        {
            "in expectation",
            "typically",
            "in practice",
            "approximately",
            "must be degenerate",
            "can be relaxed",
            "assume instead",
        };

        public void Validate(FrameContract contract, string candidateText, List<Violation> violations)
        {
            if (contract.mode == FrameMode.Ideate) return;

            // If candidate introduces drift markers without explicit disclosure tag, flag.
            // v0.1 disclosure marker convention: "[ASSUMPTION_CHANGE]"
            bool disclosed = TextScan.ContainsOrdinalIgnoreCase(candidateText, "[ASSUMPTION_CHANGE]");

            foreach (var m in DriftMarkers)
            {
                if (TextScan.ContainsOrdinalIgnoreCase(candidateText, m) && !disclosed)
                {
                    violations.Add(new Violation(ViolationCode.AssumptionDrift, details: $"Possible silent repair marker found: '{m}'."));
                }
            }
        }
    }
}
```

## 5) RewriteDirector

```csharp
using System;
using System.Collections.Generic;
using System.Text;

namespace Oan.Core.Constitution
{
    public sealed class RewriteDirector
    {
        public RewriteDirective Direct(FrameContract contract, ViolationReport report)
        {
            var sb = new StringBuilder(512);

            // Minimal instructions: remove violations, preserve meaning, satisfy obligations.
            sb.Append("Rewrite the prior output to satisfy the contract. Preserve semantics unless contract requires refusal.\n");

            if (contract.mode != FrameMode.Ideate)
            {
                sb.Append("- No questions at the end.\n");
                sb.Append("- Remove invitations/hooks and all forbidden phrases.\n");
                sb.Append("- No metaphors.\n");
                sb.Append("- Do not modify assumptions. If inconsistent, output: INCONSISTENT + Minimal Conflicting Subset.\n");
            }

            // If missing conclusion, require one.
            bool missingConclusion = false;
            foreach (var v in report.violations)
            {
                if (v.code == ViolationCode.MissingObligation)
                    missingConclusion = true;
            }
            if (missingConclusion)
            {
                sb.Append("- Must include exactly one conclusion token: PROVED, DISPROVED, INCONSISTENT, or UNDEFINED.\n");
            }

            // Deterministic directive id from contractId + violationsHash
            string seed = $"contract={contract.contractId}\nviolations={report.violationsHashHexLower}\n";
            string directiveId = Determinism.Sha256HexLower(seed);

            return new RewriteDirective(directiveId, sb.ToString());
        }
    }
}
```

## 6) Orchestrator (wraps a generator)

Generator is a callback you provide. In production it can call your model; in tests it can be a stub.

```csharp
using System;
using System.Collections.Generic;
using System.Globalization;
using System.Threading;
using System.Threading.Tasks;

namespace Oan.Core.Constitution
{
    public delegate Task<string> GeneratorAsync(string prompt, CancellationToken ct);

    public sealed class ConstitutionalOrchestrator
    {
        private readonly ContractCompiler _compiler;
        private readonly CompositeValidator _validator;
        private readonly RewriteDirector _director;
        private readonly ITelemetrySink _telemetry;

        public ConstitutionalOrchestrator(
            ContractCompiler compiler,
            CompositeValidator validator,
            RewriteDirector director,
            ITelemetrySink telemetry)
        {
            _compiler = compiler;
            _validator = validator;
            _director = director;
            _telemetry = telemetry;
        }

        public async Task<string> RunAsync(
            FrameMode mode,
            IReadOnlyList<Assumption> assumptions,
            string policyProfileId,
            string prompt,
            string configId,
            GeneratorAsync generator,
            int maxRewritePasses,
            CancellationToken ct)
        {
            var contract = _compiler.Compile(mode, assumptions, policyProfileId, maxRewritePasses);

            // Deterministic runId based on config+contract+prompt hash (NOT time)
            string promptHash = Determinism.Sha256HexLower(prompt);
            string runId = Determinism.Sha256HexLower($"config={configId}\ncontract={contract.contractId}\nprompt={promptHash}\n");

            string current = await generator(prompt, ct);
            int rewrites = 0;

            for (int attempt = 0; attempt <= contract.maxRewritePasses; attempt++)
            {
                string candHash = Determinism.Sha256HexLower(current);
                var report = _validator.Validate(contract, current);

                bool accepted = report.IsClean;

                var metrics = ComputeMetrics(contract, current, report, rewrites, accepted);

                _telemetry.Append(new RunRecord(
                    runId: runId,
                    configId: configId,
                    contractId: contract.contractId,
                    attemptIndex: attempt,
                    candidateHash: candHash,
                    violationsHash: report.violationsHashHexLower,
                    accepted: accepted,
                    rewritePassesUsed: rewrites,
                    metrics: metrics,
                    tickUtc: DateTimeOffset.UtcNow.ToUnixTimeSeconds()
                ));

                if (accepted)
                    return current;

                if (attempt == contract.maxRewritePasses)
                    break;

                var directive = _director.Direct(contract, report);
                rewrites++;

                // Rewrite prompt: keep deterministic wrapper
                string rewritePrompt =
                    "=== CONSTITUTIONAL REWRITE ===\n" +
                    "ContractId: " + contract.contractId + "\n" +
                    "DirectiveId: " + directive.directiveId + "\n" +
                    "Instructions:\n" + directive.instructions + "\n" +
                    "=== PRIOR OUTPUT ===\n" + current + "\n";

                current = await generator(rewritePrompt, ct);
            }

            // Failed to obtain compliant output. Return last output (or throw).
            // v0.1: throw, to force callers to handle explicitly.
            throw new InvalidOperationException($"CBM: failed to reach compliance after {contract.maxRewritePasses} rewrites. runId={runId}");
        }

        private static RunMetrics ComputeMetrics(
            FrameContract contract,
            string text,
            ViolationReport report,
            int rewriteCycles,
            bool accepted)
        {
            // v0.1: crude approximations for rates; later replace with tokenization-based.
            int len = Math.Max(1, text.Length);

            int hookCount = 0;
            int metaphorCount = 0;
            int driftCount = 0;
            int obligationViolations = 0;

            foreach (var v in report.violations)
            {
                switch (v.code)
                {
                    case ViolationCode.Hook: hookCount++; break;
                    case ViolationCode.Metaphor: metaphorCount++; break;
                    case ViolationCode.AssumptionDrift: driftCount++; break;
                    case ViolationCode.MissingObligation:
                    case ViolationCode.InvalidConclusion:
                    case ViolationCode.InconsistentMissingMinimalSubset:
                        obligationViolations++;
                        break;
                }
            }

            double hookRate = hookCount / (double)len;
            double metaphorRate = metaphorCount / (double)len;
            double assumptionDriftRate = driftCount / (double)len;

            // Obligation pass rate: 1 if no obligation-related violations else 0 (v0.1)
            double obligationPassRate = (obligationViolations == 0) ? 1.0 : 0.0;

            double flowScore = (accepted ? 1.0 : 0.0)
                               - 0.1 * rewriteCycles
                               - 0.2 * assumptionDriftRate
                               - 0.2 * hookRate;

            return new RunMetrics(
                hookRate: hookRate,
                metaphorRate: metaphorRate,
                assumptionDriftRate: assumptionDriftRate,
                obligationPassRate: obligationPassRate,
                rewriteCycles: rewriteCycles,
                accepted: accepted,
                flowScore: flowScore
            );
        }
    }
}
```

```csharp
// src/Oan.Cbm/Validators/ConclusionTerminalValidator.cs
using System;
using System.Collections.Generic;

namespace Oan.Core.Constitution
{
    /// <summary>
    /// Ensures the conclusion token is the last non-empty line (AUDIT/SPEC).
    /// Prevents "proof then hook" behavior.
    /// </summary>
    public sealed class ConclusionTerminalValidator : IValidator
    {
        public void Validate(FrameContract contract, string candidateText, List<Violation> violations)
        {
            if (contract.mode == FrameMode.Ideate) return;

            var last = LastNonEmptyLine(candidateText);
            if (last == null)
            {
                violations.Add(new Violation(ViolationCode.MissingObligation, details: "Empty output; missing conclusion token."));
                return;
            }

            // Must be exactly one of the allowed tokens (case-insensitive),
            // but must appear as the entire last non-empty line.
            if (!IsConclusionToken(last))
            {
                // If a conclusion token exists earlier, this is "non-terminal conclusion"
                // If none exists, it's missing obligation (handled elsewhere too)
                var found = FindAnyConclusionToken(candidateText);
                if (found != null)
                {
                    violations.Add(new Violation(
                        ViolationCode.InvalidConclusion,
                        details: $"Conclusion token '{found}' is not the last non-empty line. Last non-empty line is '{Truncate(last, 80)}'."));
                }
                // else let ObligationValidator raise MissingObligation; do not duplicate here.
            }
        }

        private static string? LastNonEmptyLine(string text)
        {
            if (string.IsNullOrEmpty(text)) return null;

            // Normalize line endings logically (do not allocate too much)
            var lines = text.Split('\n');
            for (int i = lines.Length - 1; i >= 0; i--)
            {
                var line = lines[i].Trim();
                if (line.Length == 0) continue;
                // Remove trailing \r if present from Windows line endings
                if (line.EndsWith("\r", StringComparison.Ordinal))
                    line = line.Substring(0, line.Length - 1).Trim();
                if (line.Length == 0) continue;
                return line;
            }
            return null;
        }

        private static bool IsConclusionToken(string line)
        {
            return line.Equals("PROVED", StringComparison.OrdinalIgnoreCase)
                || line.Equals("DISPROVED", StringComparison.OrdinalIgnoreCase)
                || line.Equals("INCONSISTENT", StringComparison.OrdinalIgnoreCase)
                || line.Equals("UNDEFINED", StringComparison.OrdinalIgnoreCase);
        }

        private static string? FindAnyConclusionToken(string text)
        {
            if (TextScan.ContainsOrdinalIgnoreCase(text, "PROVED")) return "PROVED";
            if (TextScan.ContainsOrdinalIgnoreCase(text, "DISPROVED")) return "DISPROVED";
            if (TextScan.ContainsOrdinalIgnoreCase(text, "INCONSISTENT")) return "INCONSISTENT";
            if (TextScan.ContainsOrdinalIgnoreCase(text, "UNDEFINED")) return "UNDEFINED";
            return null;
        }

        private static string Truncate(string s, int max)
        {
            if (s.Length <= max) return s;
            return s.Substring(0, max) + "…";
        }
    }
}
```

```csharp
// src/Oan.Cbm/Validators/QuantifierDriftValidator.cs
using System;
using System.Collections.Generic;

namespace Oan.Core.Constitution
{
    /// <summary>
    /// Flags output that weakens quantifiers / scope (e.g., FOR_ALL -> IN_EXPECTATION)
    /// unless the output explicitly discloses assumption change with "[ASSUMPTION_CHANGE]".
    ///
    /// v0.1 heuristic: if assumptions contain strong quantifiers, disallow drift phrases.
    /// </summary>
    public sealed class QuantifierDriftValidator : IValidator
    {
        // These are phrases that commonly indicate weakening from universal/pointwise claims.
        private static readonly string[] DriftPhrases =
        {
            "in expectation",
            "in the mean",
            "on average",
            "typically",
            "generally",
            "often",
            "with high probability",
            "approximately",
            "heuristic",
            "empirically",
            "in practice",
        };

        public void Validate(FrameContract contract, string candidateText, List<Violation> violations)
        {
            if (contract.mode == FrameMode.Ideate) return;

            // Disclosure marker: allow drift only if explicitly declared.
            bool disclosed = TextScan.ContainsOrdinalIgnoreCase(candidateText, "[ASSUMPTION_CHANGE]");

            // Only enforce if there exists at least one strong quantifier in assumptions.
            bool hasStrong = HasStrongQuantifier(contract.assumptions);

            if (!hasStrong) return;
            if (disclosed) return;

            foreach (var phrase in DriftPhrases)
            {
                foreach (var (s, e) in TextScan.FindAllOrdinalIgnoreCase(candidateText, phrase))
                {
                    violations.Add(new Violation(
                        ViolationCode.AssumptionDrift,
                        spanStart: s,
                        spanEnd: e,
                        details: $"Quantifier drift phrase '{phrase}' found without [ASSUMPTION_CHANGE]."));
                }
            }
        }

        private static bool HasStrongQuantifier(IReadOnlyList<Assumption> assumptions)
        {
            foreach (var a in assumptions)
            {
                var q = a.quantifier ?? "";
                // You can expand this set as your schema stabilizes.
                if (q.Equals("FOR_ALL", StringComparison.OrdinalIgnoreCase)) return true;
                if (q.Equals("POINTWISE", StringComparison.OrdinalIgnoreCase)) return true;
                if (q.Equals("HARD", StringComparison.OrdinalIgnoreCase)) return true;
            }
            return false;
        }
    }
}
```

* `remove_terminal_conclusion`: move conclusion token away from last non-empty line or delete it
* `quantifier_weaken_for_all_to_expectation`: insert “in expectation” / “on average” phrases
* `insert_assumption_repair_phrase`: insert “assume instead …” / “relax …”
* `remove_mcs_marker_when_inconsistent`: delete “Minimal Conflicting Subset” marker
* `inject_anthro_self_claim`: insert “I feel / I want / I decided …” without tags

This mutation layer is your *synthetic adversary*.

## 2) NDJSON → JSONL converter (deterministic)

This assumes your CBM telemetry NDJSON includes enough fields to link:

* contract dump (or contractId + stored contract text)
* candidate text
* violations (or violations hash + stored violations text)
* rewrite directive text

If you don’t yet store candidate/directive bodies in telemetry, the simplest approach is to add a **sidecar store** in CBM Lab:

* `runs.ndjson` (RunRecord)
* `blobs/` folder keyed by hash:

  * `{candidateHash}.txt`
  * `{directiveId}.txt`
  * `{violationsHash}.txt`
  * `{contractId}.txt`

Here’s a converter that expects that blob layout.

### `convert_ndjson_to_jsonl.py`

```python
import argparse
import json
import os
from typing import Dict, Any, Iterable, Tuple

def read_lines(path: str) -> Iterable[str]:
    with open(path, "r", encoding="utf-8") as f:
        for line in f:
            line = line.strip()
            if line:
                yield line

def load_blob(blobs_dir: str, name: str) -> str:
    p = os.path.join(blobs_dir, name)
    with open(p, "r", encoding="utf-8") as f:
        return f.read()

def stable_key(rec: Dict[str, Any]) -> Tuple[str, str, int]:
    return (rec.get("contractId",""), rec.get("runId",""), int(rec.get("attemptIndex", 0)))

def main():
    ap = argparse.ArgumentParser()
    ap.add_argument("--runs", required=True, help="Path to CBM runs NDJSON (RunRecord lines).")
    ap.add_argument("--blobs", required=True, help="Directory containing contract/candidate/violations/directive blobs.")
    ap.add_argument("--out_dir", required=True, help="Output directory for JSONL files.")
    args = ap.parse_args()

    os.makedirs(args.out_dir, exist_ok=True)

    # Collect attempt records and sort deterministically
    records = []
    for line in read_lines(args.runs):
        records.append(json.loads(line))
    records.sort(key=stable_key)

    # Output heads
    rewrite_path = os.path.join(args.out_dir, "rewrite_directive.jsonl")
    vio_path = os.path.join(args.out_dir, "violation_report.jsonl")

    rewrite_out = open(rewrite_path, "w", encoding="utf-8")
    vio_out = open(vio_path, "w", encoding="utf-8")

    for r in records:
        # Only use attempts where there were violations and a rewrite occurred/was requested.
        accepted = bool(r.get("accepted", False))
        violations_hash = r.get("violationsHash")
        candidate_hash = r.get("candidateHash")
        contract_id = r.get("contractId")

        # Load contract/candidate/violations from blobs keyed by their ids/hashes
        # Naming convention (example):
        #   contract_{contractId}.txt
        #   candidate_{candidateHash}.txt
        #   violations_{violationsHash}.txt
        contract_txt = load_blob(args.blobs, f"contract_{contract_id}.txt")
        candidate_txt = load_blob(args.blobs, f"candidate_{candidate_hash}.txt")
        violations_txt = load_blob(args.blobs, f"violations_{violations_hash}.txt")

        mode = "UNKNOWN"
        # If contract blob is canonical "key=value" lines, you can parse mode from it:
        for ln in contract_txt.splitlines():
            if ln.startswith("mode="):
                mode = ln.split("=", 1)[1].strip()
                break

        # If not accepted, we assume a rewrite directive exists for this attempt
        # Naming convention:
        #   directive_{runId}_{attemptIndex}.txt  (or directive_{directiveId}.txt)
        directive_txt = None
        if not accepted:
            run_id = r.get("runId")
            attempt = int(r.get("attemptIndex", 0))
            # try common patterns
            cand = [
                f"directive_{run_id}_{attempt}.txt",
                f"directive_{run_id}_{attempt+1}.txt",  # if you store "next directive"
            ]
            for c in cand:
                p = os.path.join(args.blobs, c)
                if os.path.exists(p):
                    directive_txt = load_blob(args.blobs, c)
                    break

        # Head A: rewrite_directive
        if directive_txt is not None:
            obj = {
                "task": "rewrite_directive",
                "mode": mode,
                "contract": contract_txt,
                "candidate": candidate_txt,
                "violations": violations_txt,
                "target": directive_txt
            }
            rewrite_out.write(json.dumps(obj, ensure_ascii=False) + "\n")

        # Head B: violation_report (always useful)
        obj2 = {
            "task": "violation_report",
            "mode": mode,
            "contract": contract_txt,
            "candidate": candidate_txt,
            "target": violations_txt
        }
        vio_out.write(json.dumps(obj2, ensure_ascii=False) + "\n")

    rewrite_out.close()
    vio_out.close()
    print(f"Wrote:\n- {rewrite_path}\n- {vio_path}")

if __name__ == "__main__":
    main()
```

### `train_sft_qlora.py`

```python
import argparse
import json
from datasets import load_dataset
from transformers import AutoTokenizer, AutoModelForCausalLM, TrainingArguments
from transformers import DataCollatorForLanguageModeling, Trainer
from peft import LoraConfig, get_peft_model, prepare_model_for_kbit_training

import torch

def format_prompt_and_target(example):
    prompt = (
        "TASK: rewrite_directive\n"
        f"MODE: {example.get('mode','UNKNOWN')}\n"
        "CONTRACT:\n" + example["contract"] + "\n"
        "CANDIDATE:\n" + example["candidate"] + "\n"
        "VIOLATIONS:\n" + example["violations"] + "\n"
        "OUTPUT:\n"
    )
    target = example["target"]
    return {"text": prompt + target}

def main():
    ap = argparse.ArgumentParser()
    ap.add_argument("--model", required=True, help="Base model id/path (gpt-oss-20b).")
    ap.add_argument("--data", required=True, help="Path to rewrite_directive.jsonl")
    ap.add_argument("--out", required=True, help="Output dir")
    ap.add_argument("--max_len", type=int, default=2048)
    ap.add_argument("--batch", type=int, default=1)
    ap.add_argument("--grad_accum", type=int, default=16)
    ap.add_argument("--lr", type=float, default=2e-4)
    ap.add_argument("--epochs", type=int, default=1)
    args = ap.parse_args()

    ds = load_dataset("json", data_files={"train": args.data})["train"]
    ds = ds.map(format_prompt_and_target, remove_columns=ds.column_names)

    tokenizer = AutoTokenizer.from_pretrained(args.model, use_fast=True)
    if tokenizer.pad_token is None:
        tokenizer.pad_token = tokenizer.eos_token

    def tokenize(batch):
        return tokenizer(
            batch["text"],
            truncation=True,
            max_length=args.max_len,
            padding=False,
        )

    tokenized = ds.map(tokenize, batched=True, remove_columns=["text"])

    # Load in 4-bit for QLoRA
    model = AutoModelForCausalLM.from_pretrained(
        args.model,
        load_in_4bit=True,
        torch_dtype=torch.bfloat16,
        device_map="auto"
    )

    model = prepare_model_for_kbit_training(model)

    # LoRA config: conservative defaults
    lora = LoraConfig(
        r=16,
        lora_alpha=32,
        lora_dropout=0.05,
        bias="none",
        task_type="CAUSAL_LM",
        target_modules=["q_proj","k_proj","v_proj","o_proj"]  # adjust if model uses different names
    )
    model = get_peft_model(model, lora)

    collator = DataCollatorForLanguageModeling(tokenizer=tokenizer, mlm=False)

    targs = TrainingArguments(
        output_dir=args.out,
        per_device_train_batch_size=args.batch,
        gradient_accumulation_steps=args.grad_accum,
        learning_rate=args.lr,
        num_train_epochs=args.epochs,
        logging_steps=10,
        save_steps=200,
        save_total_limit=2,
        bf16=True,
        optim="paged_adamw_8bit",
        lr_scheduler_type="cosine",
        warmup_ratio=0.03,
        report_to=[],
    )

    trainer = Trainer(
        model=model,
        args=targs,
        train_dataset=tokenized,
        data_collator=collator,
    )

    trainer.train()
    trainer.save_model(args.out)
    tokenizer.save_pretrained(args.out)

if __name__ == "__main__":
    main()
```

**Notes for Spark reality:**

* Start with `max_len=2048` to keep iteration fast.
* Your CBM prompts can get huge; cap aggressively at first.
* If the `target_modules` names differ for gpt-oss-20b, you’ll adjust to its attention projection module names (quickly discovered by printing `model`).

## 2) Implement an append-only filesystem blob store

This does two important things:

* **Create new only** (never overwrite)
* **Verify** if already exists (same content hash) to catch accidental nondeterminism

```csharp
// src/Oan.Cbm/Telemetry/FileBlobStore.cs
using System;
using System.IO;
using System.Text;

namespace Oan.Core.Constitution
{
    public sealed class FileBlobStore : IBlobStore
    {
        private readonly string _dir;

        public FileBlobStore(string dir)
        {
            _dir = dir;
            Directory.CreateDirectory(_dir);
        }

        public bool Exists(string blobName)
        {
            var path = Path.Combine(_dir, blobName);
            return File.Exists(path);
        }

        public string Read(string blobName)
        {
            var path = Path.Combine(_dir, blobName);
            return File.ReadAllText(path, Encoding.UTF8);
        }

        public void PutIfAbsent(string blobName, string content)
        {
            var path = Path.Combine(_dir, blobName);

            // Ensure parent dir exists (blobName may include subfolders)
            var parent = Path.GetDirectoryName(path);
            if (!string.IsNullOrEmpty(parent))
                Directory.CreateDirectory(parent);

            if (File.Exists(path))
            {
                // Verify same content (determinism guard)
                var existing = File.ReadAllText(path, Encoding.UTF8);
                if (!string.Equals(existing, content, StringComparison.Ordinal))
                {
                    string a = Determinism.Sha256HexLower(existing);
                    string b = Determinism.Sha256HexLower(content);
                    throw new InvalidOperationException(
                        $"BLOB_CONFLICT: blob '{blobName}' already exists with different content. existing={a} new={b}");
                }
                return;
            }

            // Atomic create-new
            using var fs = new FileStream(path, FileMode.CreateNew, FileAccess.Write, FileShare.Read);
            using var sw = new StreamWriter(fs, new UTF8Encoding(encoderShouldEmitUTF8Identifier: false));
            sw.Write(content);
        }
    }
}
```

## 3) Add a “CBM artifact writer” that standardizes names

This is the piece that ensures your converter script can count on filenames.

Naming conventions (simple, stable):

* `contract_{contractId}.txt`
* `candidate_{candidateHash}.txt`
* `violations_{violationsHash}.txt`
* `directive_{runId}_{attemptIndex}.txt`

```csharp
// src/Oan.Cbm/Telemetry/CbmArtifactWriter.cs
using System;

namespace Oan.Core.Constitution
{
    public sealed class CbmArtifactWriter
    {
        private readonly IBlobStore _blobs;

        public CbmArtifactWriter(IBlobStore blobs)
        {
            _blobs = blobs;
        }

        public void WriteContract(FrameContract contract)
        {
            string canonical = Determinism.CanonicalizeContract(contract);
            string name = $"contract_{contract.contractId}.txt";
            _blobs.PutIfAbsent(name, canonical);
        }

        public void WriteCandidate(CandidateOutput candidate)
        {
            string name = $"candidate_{candidate.hashHexLower}.txt";
            _blobs.PutIfAbsent(name, candidate.rawText);
        }

        public void WriteViolations(ViolationReport report)
        {
            // Store canonicalized violations text, not raw list, so it’s stable.
            // (This also makes training cleaner.)
            string canonical = Determinism.CanonicalizeViolations(report.violations);
            string name = $"violations_{report.violationsHashHexLower}.txt";
            _blobs.PutIfAbsent(name, canonical);
        }

        public void WriteDirective(string runId, int attemptIndex, RewriteDirective directive)
        {
            // Store directive instructions only (v0.1). You can expand later.
            // Keep deterministic: never embed timestamps.
            string body =
                "directiveId=" + directive.directiveId + "\n" +
                "requiredConclusion=" + (directive.requiredConclusion?.ToString() ?? "null") + "\n" +
                "instructions=\n" + directive.instructions;

            string name = $"directive_{runId}_{attemptIndex}.txt";
            _blobs.PutIfAbsent(name, body);
        }
    }
}
```

## 4) Wire this into the Orchestrator

We’ll add an optional `CbmArtifactWriter` parameter to the orchestrator. If provided, it writes blobs at each step.

### Update `ConstitutionalOrchestrator`

```csharp
// src/Oan.Cbm/ConstitutionalOrchestrator.cs (diff-style conceptual update)
using System;
using System.Collections.Generic;
using System.Threading;
using System.Threading.Tasks;

namespace Oan.Core.Constitution
{
    public sealed class ConstitutionalOrchestrator
    {
        private readonly ContractCompiler _compiler;
        private readonly CompositeValidator _validator;
        private readonly RewriteDirector _director;
        private readonly ITelemetrySink _telemetry;
        private readonly CbmArtifactWriter? _artifacts; // NEW

        public ConstitutionalOrchestrator(
            ContractCompiler compiler,
            CompositeValidator validator,
            RewriteDirector director,
            ITelemetrySink telemetry,
            CbmArtifactWriter? artifacts = null) // NEW
        {
            _compiler = compiler;
            _validator = validator;
            _director = director;
            _telemetry = telemetry;
            _artifacts = artifacts;
        }

        public async Task<string> RunAsync(
            FrameMode mode,
            IReadOnlyList<Assumption> assumptions,
            string policyProfileId,
            string prompt,
            string configId,
            GeneratorAsync generator,
            int maxRewritePasses,
            CancellationToken ct)
        {
            var contract = _compiler.Compile(mode, assumptions, policyProfileId, maxRewritePasses);

            // Persist contract blob once per run
            _artifacts?.WriteContract(contract);

            string promptHash = Determinism.Sha256HexLower(prompt);
            string runId = Determinism.Sha256HexLower($"config={configId}\ncontract={contract.contractId}\nprompt={promptHash}\n");

            string current = await generator(prompt, ct);
            int rewrites = 0;

            for (int attempt = 0; attempt <= contract.maxRewritePasses; attempt++)
            {
                string candHash = Determinism.Sha256HexLower(current);
                var candidate = new CandidateOutput(current, candHash);

                // Persist candidate blob
                _artifacts?.WriteCandidate(candidate);

                var report = _validator.Validate(contract, current);

                // Persist violations blob (canonical)
                _artifacts?.WriteViolations(report);

                bool accepted = report.IsClean;

                var metrics = ComputeMetrics(contract, current, report, rewrites, accepted);

                _telemetry.Append(new RunRecord(
                    runId: runId,
                    configId: configId,
                    contractId: contract.contractId,
                    attemptIndex: attempt,
                    candidateHash: candHash,
                    violationsHash: report.violationsHashHexLower,
                    accepted: accepted,
                    rewritePassesUsed: rewrites,
                    metrics: metrics,
                    tickUtc: DateTimeOffset.UtcNow.ToUnixTimeSeconds()
                ));

                if (accepted)
                    return current;

                if (attempt == contract.maxRewritePasses)
                    break;

                var directive = _director.Direct(contract, report);

                // Persist directive blob keyed by runId+attempt
                _artifacts?.WriteDirective(runId, attempt, directive);

                rewrites++;

                string rewritePrompt =
                    "=== CONSTITUTIONAL REWRITE ===\n" +
                    "ContractId: " + contract.contractId + "\n" +
                    "DirectiveId: " + directive.directiveId + "\n" +
                    "Instructions:\n" + directive.instructions + "\n" +
                    "=== PRIOR OUTPUT ===\n" + current + "\n";

                current = await generator(rewritePrompt, ct);
            }

            throw new InvalidOperationException($"CBM: failed to reach compliance after {contract.maxRewritePasses} rewrites. runId={runId}");
        }

        // ComputeMetrics unchanged...
    }
}
```
